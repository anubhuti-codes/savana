Savana – Fashion Store Website

Savana is a responsive front-end fashion e-commerce website developed using HTML5, CSS3, and JavaScript. The project simulates a modern online shopping experience with dynamic product rendering, interactive cart functionality, live search filtering, quantity management, automatic total calculation, and responsive UI design.

The website is inspired by modern fashion platforms and focuses on providing smooth user interaction through clean design and dynamic JavaScript functionality.

Features
Dynamic product display
Sale products section
Add to cart functionality
Quantity increase/decrease controls
Real-time subtotal and total calculation
Live search filtering
Toast notifications
Responsive design for all devices
Newsletter subscription form
